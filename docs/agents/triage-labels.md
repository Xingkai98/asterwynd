# Triage labels

Matt Pocock skills 使用五个标准 triage 角色。本文件把这些角色映射到本仓库 GitHub Issues 中实际使用的 label 字符串。

| Matt Pocock role | 本仓库 label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求已清楚，可交给 AFK agent 执行 |
| `ready-for-human` | `ready-for-human` | 需要人类实现或判断 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个角色，例如“apply the AFK-ready triage label”，使用本表右侧对应的 label。

如果未来仓库实际 label 发生变化，只更新本表右侧列，不要让 skill 创建一套重复标签。
